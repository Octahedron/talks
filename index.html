Cotopaxi: Desenvolvendo Java web para nuvem

Danilo Queiroz e Vítor Avelino

V ENSOL - Encontro Software Livre da Paraíba - 23/07/2011

O Cotopaxi é um framework Java para desenvolvimento web com foco em ambientes de computação em nuvem sob licença LGPLv3. Ele foi desenvolvido para ser um framework simples e altamente extensível. Um dos seus principais objetivos é garantir robustez e escalabilidade que são fundamentais nos dias de hoje.
